# 👀 项目概览

-   只需把音频文件分享给应用，就可以从任何位置播放歌曲，无需下载！
-   可以在所有音乐 / 专辑 / 艺术家中轻松搜索！
-   速度非常快，响应流畅！
-   不需要任何多余权限！
-   支持 Material 3 / You & Monet 主题（包含 AMOLED 模式）！
-   支持文件夹黑名单。
-   精美的横屏界面！
-   标签（Tag）编辑器！
-   支持播放列表！
-   让你变得更可爱！
